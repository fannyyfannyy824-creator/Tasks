A. Недовольные зрители
B. Отжимания (префикс)

A. Недовольные зрители

Условие
Дано число n — количество зрителей, сидящих в ряд. Каждому зрителю соответствует номер его места. Также задано число k — допустимый предел недовольства.
Недовольство зрителя
  • Если зритель не с краю, он считается недовольным, если оба его соседа сидят на местах с номерами меньше, чем у него.
  • Если зритель сидит с краю, он считается недовольным, если номер места его соседа меньше его собственного.
Проверка рассадки
Введите: 
n k
a1 a2 ... an
где a — номера мест в порядке рассадки.

B. Отжимания по дням (префиксные суммы)

Условие
Дано число n — количество записей. Каждая запись содержит два числа d и c.
Число d (1 ≤ d ≤ 7) — номер дня. Число c — общее количество отжиманий, выполненных за все дни с 1-го по d-й включительно.
Гарантируется, что данные корректны и неубывающие по дням.
Требуется:
Восстановить и вывести 7 чисел — количество отжиманий, выполненных в каждый день отдельно.
